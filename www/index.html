How to Use:

The best way to the use widget is to use an iframe to embed it onto your site.

<iframe src="http://curse.modmuss50.me/widget/233564" width="800" height="170" scrolling="no" seamless="seamless" style="border: none;"></iframe>

The number after /widget/ is the CurseForge project ID. This can be found on your CurseForge project page on the right sidebar right up at the top.

Remember that the first time a project is requested it will take slightly longer to load while the data is retrieved from CurseMeta. Project data is cached for 30 mins.

The downloader counter will count up, simulating a live download counter. This is NOT a live download counter, but is based on the last 30 days of downloads on the project and then averaged out. This can be disabled with a flag, see below.

All Flags:

These various flags when appended to the end of the URL, will change things about the widget. Include a ? before the first flag you add, and a & before any additional flags you add.

simulateDownloadCount

This flag disables the simulated "live" download counter. This flag only accepts true and false.

http://curse.modmuss50.me/widget/233564?simulateDownloadCount=false

accentColor

By default, the widget will pull a vibrant accent color from the image if it can. You can force the widget to use a certain accent color by adding a hex color code in this flag. Be sure not to use a # in the URL, as that will not work.

http://curse.modmuss50.me/widget/233564?accentColor=F78

backgroundColor

This flag can be used to set the normally transparent background to any color you wish. (Supports 'transparent')

http://curse.modmuss50.me/widget/233564?backgroundColor=F00

normalTextColor

This flag can be used to set the text color for everything outside of the buttons.

http://curse.modmuss50.me/widget/233564?normalTextColor=00F

overrideButtonTextColor

This flag can be used to override the color of the text on the Download button. Normally either white or black is automagically selected based on if the background accent color is dark or light.

http://curse.modmuss50.me/widget/233564?overrideButtonTextColor=FF0

buttonShadowColor

This flag sets the shadow color of the Download button, normally a grey color. (Supports 'transparent')

http://curse.modmuss50.me/widget/233564?buttonShadowColor=F00

darkTheme

This flag enables the dark theme. Various colors can still be overriden with the other flags, as this is checked before the others. This flag only accepts true and false.

http://curse.modmuss50.me/widget/233564?darkTheme=true

directDownload

Normally, the Download button links to the download page of the latest version. This flag will make it a direct download. This flag only accepts true and false.

http://curse.modmuss50.me/widget/233564?directDownload=true

widgetTemplate

This flag changes the html template that the widget uses. The default is horizontal but there are a couple other options you can enable with this flag.

vertical: You must set the dimensions of the iframe to a width of 220 and a height of 420 for this template. 

http://curse.modmuss50.me/widget/233564?widgetTemplate=vertical

compact: You must set the dimensions of the iframe to a width of 460 and a height of 160 for this template. 

http://curse.modmuss50.me/widget/233564?widgetTemplate=compact
